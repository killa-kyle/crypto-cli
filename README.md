# crypto
![crypto screenshot](screenshot.png)

get live crypto prices straight from the terminal


## install

 - ```npm install && npm install-global```

## use
 - launch from terminal with ```crypto```
 - this is set in package.json "bin" option
 ```  
    "bin": {
        "crypto": "./index.js"
    },
  ```

## dev

 - ```npm run dev```
